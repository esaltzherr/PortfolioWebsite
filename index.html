Game Projects

Dark Throne

Dark Throne is a 2D platforming metroidvania where a knight is trapped in a dungeon and must fight and climb their way out, with a main focus on combat and movement.

My Contributions: Lead programmer and project lead, developed player movement, abilities, backend systems including saving and scene management, Steam integration.

Extended Game Description: The game features challenging boss fights, intricate level design, and a deep narrative that keeps players engaged throughout. Players unlock new abilities and weapons as they progress, providing a sense of growth and exploration.

Technical Specifications: Built using Unity and C#. Designed a custom save system.

Team and Development Details: Developed over 6 months by a team of 5 to create a cohesive game to be submitted to Steam.

Links: Play on Steam | GitHub Repository

Where's my gun?

This is a wave-surviving game where after each wave you get to upgrade an ability you have, but the ability that you don't choose to upgrade will be disabled for the wave, so you need to be careful with balancing what you truly want.

My Contributions: Player mechanics developer; implemented movement, combat, and abilities.

Extended Game Description: The game features various enemy types, a progression system, and a unique upgrade mechanic that adds strategic depth to each playthrough through the use of massive tradoffs.

Technical Specifications: Made in JavaScript's Phaser Framework.

Team and Development Details: Developed as a 3-person team over 2 weeks.

Links: Play on Browser | GitHub Repository
